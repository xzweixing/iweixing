百度无线，免费建手机站点

发表于2012-04-14|更新于2024-08-30|tech
|阅读量:

今天看到百度无线–移动开放平台可以免费建设自己的手机站点，而且是很傻瓜的方法，十分钟就能搞定，于是乎给我的博客建了一个手机站点。 其实原理很简单，就是自己创建一个网站的二级域名，然后cname到百度的yizhan.com，百度通过自己的技术将你的网站转换成手机能访问的网站，就是这样，同时百度还提供手机站点的优化方法，请看百度的手机版页面效果优化（点击下载）。

下面是百度对手机网站建设的一些解释：

百度移动建站服务，致力于推动传统PC 站点的移动化工作，希望 不第三方PC 站点一起共同建设丰富的移动互联网世界。 百度标注协议（即Open TC 网页标注协议）是提供给第三方PC 站 点的开放协议。目的是让支持该协议的第三方PC 页面获得更好的移劢 建站效果，迚而提升手机版页面在百度移动搜索的展现效果和排名。 百度标注协议通过网站主动在PC 网页的源文件中的特定位置嵌入 百度标注协议标记，通过解析这些标记所含的诧义信息，百度移劢建站 服务对源PC 网页的处理过程会按照第三方网站的意愿迚行，实现其对 转码生成的手机版页面可控。 该协议采用HTML 标注的形式，非侵入式，丌会对原页面的功能不 展现效果造成仸何影响。标记协议可分为页面标记和块标记两类描述信 息。 协议中的属性取值部分采用在PC 网页的PC 技术中被广泛应用的 JSON 格式。一方面，方便使用者快速生成；另一方面，JSON 作为高 度灵活可扩展的格式，为后续协议的升级扩展预留了足够的空间。 JSON(JavaScript Object Notation) 是一种轻量级的数据交换格式。 页面标注协议将协议标注的内容以JSON 格式描述，并嵌入到网页的 HTML 注释中。被嵌入的标注一般对其所在文本位置后面一段范围内的 网页内容起作用。
